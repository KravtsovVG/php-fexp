PHP File Exchange Platform (PHP-FEXP)
=====================================

This directory contains the (dynamic) data that are generated and used by the
PHP File Exchange Platform.

Should you need to place this directory elsewhere, please update the 'dir_files'
and 'dir_logs' configuration parameters accordingly (see the sample 'config.php'
for further information).

ATTENTION: This directory MUST be writable (and readable) by PHP!

CRITICAL: THIS DIRECTORY MUST NOT BE ACCESSIBLE FROM THE WEB!!!
